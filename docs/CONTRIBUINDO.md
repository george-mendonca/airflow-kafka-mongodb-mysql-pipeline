# Guia de Contribuição

Obrigado pelo seu interesse em contribuir para o projeto! Aqui estão algumas diretrizes que você deve seguir ao contribuir:

## Como Contribuir

1. **Fork o repositório**: Crie uma cópia do repositório em sua conta do GitHub.

2. **Crie uma nova branch**: Sempre crie uma branch para suas alterações. Use um nome descritivo, como `feature-minha-nova-funcionalidade`.

3. **Realize suas alterações**: Faça as modificações necessárias em seu branch.

4. **Teste suas alterações**: Certifique-se de que suas alterações não quebraram nada e que estão funcionando conforme o esperado.

5. **Faça um pull request**: Quando suas alterações estiverem prontas, envie um pull request para a branch principal do repositório.

## Diretrizes de Código

- Siga as melhores práticas de codificação.
- Adicione comentários em seu código onde necessário.
- Mantenha um estilo consistente em todo o projeto.

## Reportando Problemas

Se você encontrar um problema, por favor, relate isso na seção de Issues do repositório. 

## Agradecimentos

Agradecemos suas contribuições e por ajudar a melhorar o projeto!
